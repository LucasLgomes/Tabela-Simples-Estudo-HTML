Resumo do que aprendi em HTML até agora.
Listas não ordenadas são usadas para agrupar um conjunto de itens relacionados, sem uma ordem específica. Por exemplo, uma lista de tópicos para um artigo ou uma lista de itens para uma loja. Listas ordenadas são usadas para agrupar um conjunto de itens relacionados, em uma ordem específica. Por exemplo, uma lista de passos para fazer um bolo ou uma lista de números de telefone. O atributo id é usado para identificar um elemento HTML de forma única. Isso pode ser útil para referenciar o elemento em CSS ou JavaScript. Os links são usados para conectar uma página da web a outra. Por exemplo, um link para o site de uma empresa ou um link para uma página de produto. As imagens podem ser inseridas em uma página da web para adicionar conteúdo visual. Por exemplo, uma imagem de um produto ou uma imagem de um local turístico.
